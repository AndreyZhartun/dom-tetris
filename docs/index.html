Игра окончена!
Игровое поле состоит из 12 .col-1 колонн с 12 div-ами в каждой.
Движущаяся фигура представляет собой коллекцию элементов JQuery. Падение осуществляется путем смены мест div-ов с помощью функции JQuery before().
Столкновения проверяются путем проверки наличия классов цвета с помощью функции JQuery hasClass().
Боковое движение основано на нахождении элементов на таких же вертикальных позициях в соседних колоннах с помощью функций JQuery index()&eq().
Поворот фигуры реализован на основе матрицы поворота.
