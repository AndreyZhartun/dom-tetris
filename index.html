Game Description
Made with Bootstrap & JQuery

The grid is made of 12 .col-1 columns with 12 divs each.

The moving figure is a JQuery collection of divs. Falling down is implemented by exchanging positions of divs with JQuery before().

Side movement looks up elements with the same vertical position in adjacent columns using JQuery index()&eq().

Collision logic uses .bg-color classes with JQuery hasClass().
